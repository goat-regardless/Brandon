brizz.kixx — static 3-page site

Pages:
- index.html  (landing)
- shoes.html  (shoes catalog)
- apparel.html (apparel catalog)

Replace placeholder images:
- Put shoe images at assets/shoes/shoe-01.jpg ... shoe-09.jpg
- Put apparel images at assets/apparel/apparel-01.jpg ... apparel-09.jpg
(You can also use .png; keep the same filenames.)

Vercel deploy:
- Import repo → Framework: Other → No build command → Deploy.

Local preview:
- Open index.html in a browser, or run:  python3 -m http.server 8080
